# tenpin-solitaire

An attempt at implementing Bowling Solitaire - [link](https://boardgamegeek.com/boardgame/25314/bowling-solitaire)

## Project Setup

```sh
npm install
```

### Compile and Hot-Reload for Development

```sh
npm run dev
```

### Compile and Minify for Production

```sh
npm run build
```
